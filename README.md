堆塔游戏 - 游戏说明

【游戏简介】
这是一款考验反应力和节奏感的休闲益智游戏。玩家需要通过精准的点击，将不断移动的方块一层层堆叠起来，挑战建造最高的塔楼。

【操作说明】
1. 开始游戏：
   - 点击屏幕任意位置，或按空格键开始游戏。

2. 游戏操作：
   - 游戏中，新的方块会从屏幕一侧左右移动。
   - 当方块移动到塔楼正上方时，点击屏幕（或按空格键）将其放下。

【游戏规则】
1. 堆叠判定：
   - 如果放置的方块与下方方块完全重合，为完美放置。
   - 如果放置不够精准，超出下方方块的部分会被切除并掉落。
   - 下一块方块的大小将基于被切除后剩余的大小生成，因此游戏难度会随着方块变小而增加。

2. 游戏结束：
   - 如果方块完全未能放置在下方塔楼上（即完全错开），游戏结束。
   - 游戏结束后会显示最终得分，点击“再玩一次”即可重新挑战。

3. 计分规则：
   - 每成功堆叠一层计 1 分。
   - 随着层数增加，方块移动速度会逐渐加快。
